# 🚀 CloudFlow - SaaS Landing Page

A modern, responsive SaaS landing page built using **HTML5 + CSS3**.  
CloudFlow is a fictional workflow automation platform designed to showcase a professional UI/UX layout for software companies.


## ✨ Features

- ⚡ Fully responsive design (Mobile + Desktop)
- 🎨 Modern SaaS UI/UX layout
- 🧭 Sticky navigation bar with hamburger menu
- 🧩 Multiple sections:
  - Hero section
  - Services
  - Features
  - Testimonials
  - Pricing plans
  - Contact form
- 💬 Clean typography and professional color scheme
- 🖼️ Image support for testimonials
- 🚀 Smooth animations and transitions

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox + Grid)
- Responsive Design Principles
- Pure CSS (No frameworks, no JavaScript)
